Example of Oauth2 - Resource Owner Password Credentials Grant:
In this example you can create users. Then you can ask for two types of tokens: one giving permission to the user to read and write messages and another just giving permission to the user to read messages. The tokens will be stored in a cookie. Once you have a token you can use the "Messenger application" and notice the difference between one token and the other.
For an explanation of this example please read C++ REST SDK authentication server article.
