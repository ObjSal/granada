To test POST set the "session_token_support" property to "post" in the server.conf file and restart the server application.
# cookie || get || post
session_token_support=post
To test GET set the "session_token_support" property to "get" in the server.conf file and restart the server application.
# cookie || get || post
session_token_support=get
To test with Redis, use RedisStorageSession and include RedisCacheDriver in the project.
